## 阶段 4：JavaScript 交互 ✅ 已完成

**目标**：让网页「活」起来，增加用户交互功能。

### 学习内容
- JavaScript 是什么（网页的"大脑"，处理交互逻辑）
- 如何引入 JS 文件：`<script src="js/main.js"></script>`
- `document.addEventListener('DOMContentLoaded', ...)` — 等页面加载完再执行
- `querySelector` / `querySelectorAll` — 用 CSS 选择器找元素
- `addEventListener` — 给元素绑定事件（点击、滚动、按键）
- `localStorage` — 浏览器本地存储（记住用户的深色模式偏好）
- `classList.toggle` / `classList.add` / `classList.remove` — 动态切换 CSS 类

### 新增文件

#### `js/main.js` — 2026-06-13 创建
包含三个功能模块：

**🖼️ 图片灯箱**
- 点击任意 `.card-img` 图片 → 弹出全屏大图
- 显示图片标题和描述作为说明文字
- 三种关闭方式：× 按钮 / 点击背景 / 按 ESC 键
- 打开时锁定背景滚动

**🌙 深色模式**
- 导航栏右侧增加切换按钮（🌙/☀️）
- 点击切换浅色/深色主题
- 使用 `localStorage` 记住用户选择，刷新不丢失
- 深色模式覆盖：背景、卡片、表格、文字均为暗色调

**⬆️ 回到顶部**
- 页面滚动超过 400px 时，右下角出现圆形按钮
- 点击平滑滚动回页面顶部
- 带淡入/淡出动画

### CSS 新增样式
- `.lightbox-overlay` — 灯箱遮罩层
- `.dark-mode-toggle` — 深色模式按钮
- `body.dark-mode` — 深色模式全局样式
- `.back-to-top` — 回到顶部按钮

### 关键知识点
- JS 文件放在 `</body>` 前，确保 HTML 先加载
- `position: fixed` 让灯箱覆盖整个屏幕
- `z-index: 9999` 让灯箱显示在所有内容上方
- 深色模式用 `body.dark-mode` 作为选择器前缀，一个 class 控制全局

### 🔥 Firebase 配置指南（你需要做的 Part B）

> **前提条件**：拥有一个谷歌/Gmail 账号

#### 第 1 步：创建 Firebase 项目
1. 打开浏览器，访问 **https://console.firebase.google.com**
2. 用谷歌账号登录
3. 点击 **「创建项目」**（Add project）
4. 项目名称输入：`chunxiao-art-studio`（或任意名字）
5. 是否启用 Google Analytics：**建议关闭**（更简单）
6. 点击「创建项目」，等待完成

#### 第 2 步：启用邮箱登录
1. 进入项目后，左侧菜单点击 **「身份验证」**（Authentication）
2. 点击 **「开始使用」**
3. 选择 **「电子邮件/密码」**
4. 点击 **「启用」**，然后**「保存」**

#### 第 3 步：获取配置信息
1. 左侧菜单点击 **「项目设置」**（齿轮图标 ⚙️）
2. 在「您的应用」区域，点击 **「Web」** 图标（</>）
3. 应用昵称输入：`春晓画室`
4. **勾选「同时设置 Firebase Hosting」**（后面部署会用到）
5. 点击「注册应用」
6. 你会看到一段类似这样的代码：
```js
const firebaseConfig = {
  apiKey: "AIzaSyDxxxxxxxxxxxxxxxxxxxxx",
  authDomain: "chunxiao-xxxxx.firebaseapp.com",
  projectId: "chunxiao-xxxxx",
  storageBucket: "chunxiao-xxxxx.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:xxxxxxxxxxxxxx"
};
```
7. **复制这些值**，然后点「继续控制台」

#### 第 4 步：替换配置
1. 用记事本打开 `G:\春晓画室\js\firebase-config.js`
2. 找到 `const firebaseConfig = { ... }` 部分
3. 把里面的内容**替换成你刚复制的**
4. 保存文件

#### 第 5 步：测试
1. 双击打开 `login.html`
2. 切换到「家长登录」，点击「点击注册」
3. 填写信息完成注册
4. 注册成功后会自动跳转到家长端
5. 退出后，用刚才的邮箱密码登录
